# Fedasync

The demo of [asynchronous federated optimization](https://arxiv.org/abs/1903.03934).

## Requirements

fedlab==1.1.2

## Run

standalone:

`$ cd standalone`

`$ python standalone.py`


cross_process:


`$ cd cross_process`

`$ bash quick_start.sh`

## Performance

Null
